Configuración de Servidor / GitHub Pages

Guarda tus datos de forma permanente y gratuita en la nube

¿Por qué es necesario este paso en GitHub?

GitHub Pages aloja páginas de manera estática. Al recargar la página, cualquier cambio que realices se perdería.

Para solucionar esto, esta aplicación viene pre-equipada para conectarse de forma segura a Firebase Firestore.

Pasos para activar el guardado permanente:

  1. Entra a Firebase Console y haz clic en Crear un Proyecto.
  2. En el panel lateral izquierdo, ve a Build > Authentication, haz clic en comenzar, entra a la pestaña "Sign-in method" y habilita el proveedor Anónimo (Anonymous).
  3. Ve a Build > Firestore Database y haz clic en Crear base de datos. Selecciónala en "Modo de Prueba" para que permita guardar datos sin restricciones de lectura/escritura inmediatamente.
  4. En la configuración de tu proyecto (icono de engranaje ⚙️), añade una nueva Web App (). Copia el objeto de configuración que te entregarán.
  5. Abre tu archivo index.html con cualquier editor de código, busca la sección GITHUB_FIREBASE_CONFIG (cerca de la línea 19) y pega tus credenciales allí.
